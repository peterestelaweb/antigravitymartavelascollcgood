Guía Básica de Git y GitHub

Bienvenido a tu primera página de recursos de IA y Programación. Aquí tienes los comandos básicos para empezar.

Nota: Esta página está alojada localmente en public/ai-library/git-basics.html.

1. Configuración Inicial

git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"

2. Iniciar un Repositorio

Para empezar a rastrear un proyecto:

git init

3. Guardar Cambios

El ciclo básico de trabajo es:

  1. Verificar estado: git status
  2. Añadir archivos: git add .
  3. Guardar versión: git commit -m "Descripción del cambio"

4. Subir a GitHub

git push origin main
